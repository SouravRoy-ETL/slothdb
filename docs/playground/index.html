        .ask

        Natural language, inside the shell. Type .ask at the slothdb> prompt to open an interactive NL sub-REPL. SlothDB translates English to SQL, shows it to you, and gates every run with [Y/n].

        Rules-first (COUNT, SUM, AVG, GROUP BY, TOP-N, year filters, file-source CREATEs); builds with -DSLOTHDB_ASK_MODEL=ON fall back to a local Qwen2.5-Coder GGUF (~310 MB, llama.cpp, lazy-downloaded on first use). No cloud, no API keys.

        Available in the native CLI shell, not this in-browser playground. Download from the releases page or run pip install slothdb and use the bundled shell.

        Full .ask documentation ->

        About

        SlothDB is an embedded analytical database that reads CSV, Parquet, JSON, Arrow, SQLite, and Excel directly from SQL. This playground is the full engine compiled to WebAssembly.

        Files you add stay on your machine - nothing is uploaded.
