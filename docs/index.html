A fast, in-process SQL database
for your local data files.

SlothDB is a drop-in DuckDB alternative with built-in readers for Parquet, CSV, JSON, Avro, Arrow and SQLite. 1.1×–8.6× faster than DuckDB on every query in the benchmark suite.

$ pip install slothdb
Get started View on GitHub

60-second tour

After pip install slothdb, run python -c "import slothdb; slothdb.demo()" — it generates a 100 000-row CSV, runs three queries, and prints a side-by-side against DuckDB on your own machine.

What makes it different

Same embedded model as DuckDB and SQLite — link it in, point SQL at files on disk. The defaults are different.

Seven file formats built in

Parquet, CSV, JSON, Avro, Arrow, SQLite, Excel — all in core. DuckDB needs separate extensions for Avro, SQLite and Excel.

1.1×–8.6× faster than DuckDB

On a 1M-row benchmark across 15 queries spanning every supported format. Avro SUM is 5.4×. CSV COUNT(*) is 5.1×. Parquet SUM is a tie.

Stable C ABI for extensions

Numeric error codes (ErrorCode::TABLE_NOT_FOUND = 2000) that never change. Extensions built for v0.1 keep working on v1.0.

~8 MB self-contained binary

One file on disk. No runtime dependencies. Embeds into your app, runs as a shell, or imports into Python.

No server, no import, no setup

Point SQL at the file and go. No CREATE TABLE, no COPY FROM, no extension to load first.

MIT licensed

Use it anywhere, including commercial products. Source on GitHub, 326 tests, 131 000+ assertions.

Query any file with SQL

No CREATE TABLE. No import step. Point at the file.

-- Local Parquet / CSV / JSON
SELECT region, SUM(revenue)
FROM 'sales.parquet'
WHERE year >= 2023
GROUP BY region;

-- Avro, Arrow, SQLite — no extension load needed
SELECT * FROM read_avro('events.avro') LIMIT 100;
SELECT * FROM sqlite_scan('app.db', 'users');

-- Export results to any format
COPY (SELECT * FROM 'big.csv' WHERE year >= 2024)
TO 'filtered.parquet' WITH (FORMAT PARQUET);

Install

Pick a platform. Single binary, no dependencies.

Python

pip install slothdb
python -c "import slothdb; slothdb.demo()"

Linux / macOS (curl)

curl -fsSL https://raw.githubusercontent.com/\
SouravRoy-ETL/slothdb/main/install.sh | bash

Windows

Download slothdb.exe from
github.com/SouravRoy-ETL/slothdb/releases/latest

Build from source

git clone https://github.com/SouravRoy-ETL/slothdb
cd slothdb
cmake -B build -DSLOTHDB_BUILD_SHELL=ON
cmake --build build --config Release
