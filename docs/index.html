If you outgrow one machine, use ClickHouse. If you don't, why run a cluster?

vs SQLite — if you're using it for analytics

# SQLite is row-oriented - reads every column of every row, even
# for aggregate-only queries. SlothDB is columnar + vectorized.

# On SELECT region, SUM(revenue) FROM sales over 1M rows:
SQLite:   ~3000 ms  (reads 10 columns x 1M rows)
SlothDB:  ~100 ms   (reads 2 columns x 1M rows, vectorized)

# Keep your SQLite file, query it without migrating:
SELECT * FROM sqlite_scan('app.db', 'users');

What SlothDB does not do

  • No distributed query execution · use ClickHouse at scale
  • No MVCC / multi-writer transactions · OLTP is a poor fit
  • Younger codebase · 326 tests but corners of SQL may surprise you

LEVEL 05 — CODE

five minutes with slothdb

No CREATE TABLE. No COPY FROM. Just point SQL at your files.
SQL · query any file directly
-- CSV
SELECT region, SUM(revenue) FROM 'sales.csv' GROUP BY region;

-- Parquet with a filter
SELECT * FROM read_parquet('events.parquet')
WHERE event_date > '2024-01-01';

-- JSON / NDJSON
SELECT status, COUNT(*) FROM 'api_logs.json' GROUP BY status;

-- Excel
SELECT * FROM read_xlsx('quarterly_report.xlsx');

-- Chinese / Japanese / Korean column names work unquoted
CREATE TABLE 销售 (区域 VARCHAR, 销售额 BIGINT);
SELECT 区域, SUM(销售额) FROM 销售 GROUP BY 区域;
Python · pip install slothdb
import slothdb

db = slothdb.connect()                    # in-memory
db = slothdb.connect("analytics.slothdb") # persistent

df = db.sql("""
    SELECT name, department, salary
    FROM 'employees.parquet'
    QUALIFY ROW_NUMBER() OVER (
        PARTITION BY department ORDER BY salary DESC
    ) = 1
""").fetchdf()  # pandas DataFrame
C/C++ · stable C ABI
#include "slothdb/api/slothdb.h"

slothdb_database *db; slothdb_connection *conn; slothdb_result *r;
slothdb_open("analytics.slothdb", &db);
slothdb_connect(db, &conn);
slothdb_query(conn,
    "SELECT region, SUM(revenue) FROM read_csv('sales.csv') GROUP BY region",
    &r);

for (uint64_t i = 0; i < slothdb_row_count(r); i++)
    printf("%s: %s\n", slothdb_value_varchar(r, i, 0),
                         slothdb_value_varchar(r, i, 1));

slothdb_free_result(r);
slothdb_disconnect(conn);
slothdb_close(db);
